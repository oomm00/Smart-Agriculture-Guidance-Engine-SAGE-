# Smart Agriculture Guidance Engine (SAGE)

AI-powered crop advisory chatbot for field supervisors of an organic farming collective in Uttarakhand, giving practical guidance on crop diseases, pests, and post-harvest handling in English/Hindi via the Gemini API.

## Tech Stack
- Frontend: React (Vite) + Tailwind CSS
- Backend: FastAPI
- Database: SQLite/PostgreSQL
- API: Gemini 1.5 Flash API
